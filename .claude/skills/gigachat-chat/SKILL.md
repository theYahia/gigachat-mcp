# /gigachat-chat -- Диалог с GigaChat

## Используемые инструменты

| Инструмент | Что делает |
|------------|------------|
| `chat_completion` | Генерация текста через GigaChat |

## Алгоритм

1. Разбери запрос пользователя: текст, модель, температура
2. Если модель не указана -- по умолчанию GigaChat
3. Вызови `chat_completion` с параметрами
4. Покажи результат с информацией о модели и использованных токенах

## Формат ответа

```
## Ответ GigaChat

**Модель**: GigaChat
**Ответ**: <текст ответа>
**Токены**: ввод: 15, вывод: 128, всего: 143
```

## Примеры

```
/gigachat-chat Напиши стихотворение о весне
/gigachat-chat Объясни квантовую механику GigaChat-Pro 0.3
```
